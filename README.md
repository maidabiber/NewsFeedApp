# NewsFeedApp 

NewsFeedApp aplikacija je individualni projekat razvijen u sklopu predmeta **Razvoj mobilnih aplikacija** na trećem semestru **Elektrotehničkog fakulteta Univerziteta u Sarajevu** (smjer Računarstvo i informatika). 

##  Opis projekta

Aplikacija  povezuje rad sa udaljenim API servisima i lokalnom bazom podataka, omogućavajući korisniku filtriranje sadržaja po datumu i nepoželjnim riječima, uz mogućnost pregleda sličnih objava na osnovu kategorija ili tagova. Fokus je stavljen i na offline režim rada, koji omogućava spremanje vijesti u lokalnu bazu kako bi sav sadržaj ostao dostupan korisniku i bez internet konekcije.
