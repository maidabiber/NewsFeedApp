# NewsFeedApp 

NewsFeedApp aplikacija je projekat razvijen u sklopu predmeta **Razvoj mobilnih aplikacija (RMA)** na trećem semestru, **Elektrotehnički fakultet Sarajevo** (smjer Računarstvo i informatika). 

##  Opis projekta
Aplikacija omogućava korisnicima pregled najnovijih vijesti i analizu vizuelnog sadržaja koristeći moderne Android tehnologije.
