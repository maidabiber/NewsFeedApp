# NewsFeedApp 

NewsFeedApp aplikacija je projekat razvijen u sklopu predmeta **Razvoj mobilnih aplikacija** na trećem semestru, **Elektrotehnički fakultet Sarajevo** (smjer Računarstvo i informatika). 

##  Opis projekta

Aplikacija  povezuje rad sa udaljenim API servisima i lokalnom bazom podataka, omogućavajući korisniku napredno filtriranje sadržaja po datumu i nepoželjnim riječima, uz opcije spašavanja vijesti i pregleda sličnih objava na osnovu kategorija ili tagova.
